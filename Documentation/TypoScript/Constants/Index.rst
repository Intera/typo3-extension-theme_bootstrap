TypoScript-Constants
====================

TypoScript-Constants categories
===============================

* **theme** General theme settings
* **colors** Color settings
* **font** Font setting
* **menu** menu settings
* **header** Header settings (Logo, and more)
* **footer** Footer settings (Copyright-Notice, and more)
* **pages** Definition of Page-Uids
* **container** Definition of Container-Uids
* **meta** Meta settings
* **languages** Language settings
* **socialmedia** Socialmedia settings
* **extension** Extension settings (Label must start with ``EXT:extension_key - Labeltext...``, so we're able to sort them)
